Downtown Rio Museums

Cliente: Pessoal

Data: 29 de Janeiro de 2017

Descrição

Um site contendo um mapa interativo com localização e informações sobre museus no centro do Rio de Janeiro. Este é um projeto do programa Front-End Web Developer Nanodegree da Udacity. A aplicação utiliza a API do Google Maps para gerar o mapa, API da Foursquare para fornecer informações sobre os locais e o framework Knockout para o desenvolvimento em padrão MVVM e para lidar com informações na página que estão sujeitas a sofrerem alterações.

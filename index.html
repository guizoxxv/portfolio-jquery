Reaccit

Jan, 2018

Aplicativo web de conteúdo e comentários utilizando React e Redux.

Reaccit

Cliente: Pessoal

Data: 29 de Janeiro de 2018

Descrição

Este projeto pertence ao programa Nanodegree Desenvolvedor React. O propósito deste projeto é desenvolver um aplicativo web de conteúdo e comentários utilizando React e Redux. Este aplicativo permite que os usuários postem conteúdo em categorias pré-definidas, façam comentários em suas próprias postagens e nas de outros usuários e votem nas postagens e comentários. Os usuários também podem editar e excluir postagens e comentários.

Downtown Rio Museums

Jan, 2017

Um site contendo um mapa interativo com localização e informações sobre museus no centro do Rio de Janeiro.

Downtown Rio Museums

Cliente: Pessoal

Data: 29 de Janeiro de 2017

Descrição

Um site contendo um mapa interativo com localização e informações sobre museus no centro do Rio de Janeiro. Este é um projeto do programa Front-End Web Developer Nanodegree da Udacity. A aplicação utiliza a API do Google Maps para gerar o mapa, API da Foursquare para fornecer informações sobre os locais e o framework Knockout para o desenvolvimento em padrão MVVM e para lidar com informações na página que estão sujeitas a sofrerem alterações.
